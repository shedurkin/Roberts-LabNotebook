To get started on additional Chpt 3/4 work, I first need to compile some physiologically relevant gene sets. Several Chpt 3/4 questions involve testing epigenetic \(\leftrightarrow\) transcriptomic \(\leftrightarrow\) physiological relationships. We have an existing set of host physiology metrics that have been collected, so I now want to focus on isolating relevant transcriptomic and epigenetic subsets for each of those physiological domains.

The host phys metrics we have are:

  • calcification

  • antioxidant capacity

  • respiration

  • biomass

  • protein

We already have a biomineralization gene set compiled and blasted against our 3 study species (details in timeseries_molecular/M-multi-species/scripts/31.1-biomin-blast.Rmd). This was originally done by Jill, and as far as I can tell the biomineralization gene set was acquired from Scucchia et al. (2021).

Now I want to compile similar gene sets for the other phys domains: antioxidant capacity, respiration, and growth (biomass and protein). I’d also like to compile a set for symbiosis-related genes, since Chpt 4 relates to epigenetics mediating symbiosis.

After some preliminary searching, I wasn’t able to manually find pre-compiled comprehensive gene sets for these additional phys domains (unlike the biomin source). To facilitate the time-intensive task of identifying domain-relevant genes/proteins, I used Claude Opus 4.8.

Final script: timeseries_molecular/M-multi-species/scripts/35-physiology-gene-sets.py

Approach

For now, since I want to quickly trial some preliminary analyses, I’ll just base gene identification on existing GO annotations performed using a blast of our species’ reference against the Uniprot/Swissprot db. In the future, this should be extended or replaced by manual fasta curation and proteome blast (as in DDE) (much more time consuming)

To identify domain-relevant genes, two complementary approaches will be used:

  1. Curated members: Canonical genes drawn from the marine-invertebrate / cnidarian-symbiosis literature, each tied to a citation. This is particularly useful for recovering genes whose role is established functionally even when their generic GO terms are uninformative (especially symbiosis).
  2. GO-term membership: Genes with a domain-defining GO term in the reviewed annotation. Terms are chosen to be specific to the process (e.g. antioxidant uses antioxidant/peroxidase/glutathione/redox terms, NOT the broad “response to oxidative stress”, which pulls in signalling/apoptosis).
AI usage

Claude Opus 4.8 was used in both curating genes/proteins from the literature and identifying domain-defining GO terms for all 4 of the physiological domains targeted here

A gene is included only if it has a reviewed Swiss-Prot best hit (e-value <= 1e-10) AND qualifies through one or both of the above approaches

To treat the three study species symmetrically, each set is keyed on the ortholog group and restricted to three-way orthologs (present in Apul, Peve, and Ptua). The A.pulchra functional annotation is used for calling, and the call is then applied to the full ortholog group (note that this assumes that orthology conserves function).

The growth set captures anabolic / protein-biosynthetic capacity (cytoplasmic ribosome biogenesis, translation machinery, TOR signalling; + a tight cell-proliferation core), the canonical molecular correlate of biomass/protein via the growth-rate hypothesis (Elser et al. 2003; Warner 1999). Mitochondrial/organellar translation genes are vetoed to avoid overlap with respiration.
